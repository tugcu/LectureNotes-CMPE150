Drawing Circles with Turtle

Back to the old turtle, let's draw a set of circles:

import turtle

turtle.speed(8)
turtle.pensize(5)

turtle.pencolor("Green")
turtle.circle(40)

turtle.pencolor("Blue")
turtle.circle(60)

turtle.pencolor("Orange")
turtle.circle(80)

turtle.pencolor("Pink")
turtle.circle(100)

turtle.pencolor("Red")
turtle.circle(120)

turtle.pencolor("Purple")
turtle.circle(140)

turtle.pencolor("Magenta")
turtle.circle(160)

turtle.pencolor("Cyan")
turtle.circle(180)

turtle.pencolor("Yellow")
turtle.circle(200)

turtle.done()
